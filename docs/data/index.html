Data

Data is at the heart of our teamâ€™s effort to better predict the spread of COVID-19 in the US. We have been systematically tracking data from multiple different COVID-19 prediction models since early April 2020.

Visualization

We have built an interactive visualization so you can explore recent forecasts from multiple teams. Having all these forecasts in one place enables simple comparisons between models and model synthesis for decision-makers.

Raw Forecasts

All COVID-19 Forecast Hub data is stored publicly in a structured data storage repository on GitHub. It can also be downloaded programmatically from our Zoltar API. Currently we have over 25m rows of forecast data available.

Software & tools

Our team has been developing open-source software for years to assist with forecast modeling in outbreak response. We have developed numerous packages including the FluSight R package, the d3-foresight javascript library, and the Zoltar system for forecast storage.
