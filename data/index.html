Data

We have been systematically tracking data from dozens of COVID-19 prediction models since early April 2020.

Forecast Data

Over 100m rows of COVID-19 Forecast Hub data follow a data model for probabilistic forecasts specified by quantiles. These data are stored publicly in a structured data storage repository on GitHub. They can also be downloaded programmatically from our Zoltar API.

Visualization

We have built an interactive visualization so you can explore forecasts from multiple teams. Having all these forecasts in one place enables simple comparisons between models.

Software & tools

Our team has been developing open-source software for years to assist with forecast modeling in outbreak response. We have developed the d3-foresight javascript library for visualization and the Zoltar system for forecast storage. We also have developed the covidHubUtils and covidData packages as resources for accessing data for and about forecast models.
